# AI Chatbot using NLTK

## Project Description

This project is a simple rule-based AI Chatbot developed using Python and the NLTK library. The chatbot uses regular expression pattern matching to understand user input and generate appropriate responses.

## Objectives

* Build a simple chatbot using Python.
* Use NLTK for conversational interaction.
* Implement pattern matching using regular expressions.
* Provide greetings and small talk.
* Handle unknown inputs with fallback responses.
* Create a user-friendly interface using Tkinter.

## Technologies Used

* Python
* NLTK
* Tkinter

## Features

* Greeting responses
* Small talk conversations
* Joke responses
* Fallback responses
* Dynamic user interaction
* GUI interface

## How to Run

1. Install Python.
2. Install NLTK using:

pip install nltk

3. Run the chatbot:

python gui.py
